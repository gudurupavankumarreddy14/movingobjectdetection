# Moving Object Detection

## Project Description
This project is a real-time Moving Object Detection system developed using Python and OpenCV. It detects moving objects in video streams by comparing consecutive frames, identifying motion, and highlighting detected objects with bounding boxes.

## Features
- Real-time moving object detection
- Motion tracking using frame differencing
- Bounding box detection around moving objects
- Simple and efficient implementation
- Works with webcam or video input

## Technologies Used
- Python
- OpenCV
- NumPy

## Requirements
Install required libraries:
pip install opencv-python numpy

##How to Run :
Download or clone this repository
Open terminal in project folder


##Run:
python moving.py

##Project Output:
The system detects moving objects in the camera/video feed and displays bounding boxes around detected motion.
